# Syria Electricity Calculator

A small Vite + React + TailwindCSS app for calculating electricity metrics (working title).

## Tech

- Vite
- React
- Tailwind CSS

## Setup

1. Install dependencies:

```powershell
npm install
```

2. Run the dev server:

```powershell
npm run dev
```

3. Build for production:

```powershell
npm run build
```

## Notes

- The primary color has been set in `src/index.css` to `#054239` (HSL: `171.1 85.9% 13.9%`).
- Tailwind config is in `tailwind.config.js`.
